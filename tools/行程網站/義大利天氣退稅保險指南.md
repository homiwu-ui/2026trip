## 二、退稅指南（VAT Refund）

### 義大利 VAT 稅率
| 稅率 | 適用商品 |
|---|---|
| **22% 標準稅率** | 精品、服飾、電器、化妝品、珠寶（旅人 main target） |
| 10% 減免稅率 | 飯店、餐廳、交通、食品 |
| 5% 減免稅率 | 社福服務、老年人/身障用品 |
| 4% 超減免稅率 | 民生必需品（麵包、牛奶、書籍、藥品） |

### 退稅資格
- ✅ 非歐盟居民（台灣護照符合）
- ✅ 年滿 16 歲
- ✅ 在義停留少於 6 個月
- ✅ 單筆消費 **滿 €70.01**（2023 年從 €154.95 調降）
- ❌ 服務類不適用（飯店、餐廳、交通）

### 退稅流程
```
購物 → 索取退稅單 → 離境前驗證 → 取得退稅
```

**Step 1 — 購物時**
- 到有「Tax Free」標誌的商店（Global Blue、Planet、Zapptax 等）
- 出示護照，請店員開立 **Tax Free Form**
- 確認店員填妥資料（姓名、護照號碼、地址、退稅方式）

**Step 2 — 離境前驗證（米蘭 MXP 機場）**
- ⚠️ **必須在 check-in 之前、過安檢之前** 辦理
- 行李中退稅商品要可隨時拿出檢查
- **電子驗證**：找 PABLO 自助 kiosk，掃描退稅單條碼
- **人工驗證**：到海關櫃檯蓋章（買高價品建議走人工）
- 退稅單時效：開立日起 **5 個月內**（2026 年新規，原先為 3 個月）

**Step 3 — 取得退稅**
- 驗證後到退稅公司櫃檯（Global Blue / Planet）領取：
  - **現金**：每張退稅單上限 €1,000，現場拿
  - **信用卡**：上限 €3,000，約 2–10 工作天入帳
  - **支付寶**：部分支援

### 實際退稅金額
- 雖名義 VAT 為 22%，扣除手續費後約可拿回 **12–15%**
- 舉例：買 €1,000 商品 → 約退 €120–150
- 退稅公司手續費：Global Blue 約 3–4%，Planet 類似

## 五、台灣旅遊不便險 — 完整比較

### 什麼是旅遊不便險？跟產險的關係？

| 問題 | 答案 |
|---|---|
| **不便險是誰在賣？** | **產物保險公司**（簡稱產險）。因為不便險（班機延誤、行李遺失、旅程取消）屬於「財產保險」，不是人身保險 |
| **跟壽險旅平險差在哪？** | 壽險公司賣的「旅平險」只賠身故/失能/醫療；「不便險」賠航班延誤、行李、行程取消等。**兩個是不同的險種** |
| **可以單買不便險嗎？** | ✅ 可以。但多數產險的網路方案已是「旅平+不便」套餐，保費極低，不需要特地分開買 |
| **投保限制？** | 須在 **台灣境內** 投保（出發前）。人已在國外就無法投保台灣產險不便險 |

### 台灣產險公司 申根不便險 重點比較（12天義大利）

### 航班
- 7 月為旺季，機票已訂好勿忘 **48 小時前線上 check-in**
- 米蘭 MXP 建議提前 3 小時到機場（退稅 + 安檢人潮多）

### 換匯
- 歐元（EUR），在台灣先換少量現金
- 大部分商店可刷卡（Visa/Mastercard 通行，American Express 較少）
- 可帶 **€200–300 現金** 以備不時之需（Gondola 只能付現）

### 電信
- 台灣門號漫遊：中華/遠傳/台哥大均有歐洲漫遊方案
- 或到義大利買預付卡（TIM / Vodafone / WindTre）
- 或租 WiFi 機

### 安全
- 旅遊旺季扒手猖獗，尤其在：
  - 羅馬 Termini 車站、地鐵、西班牙階梯
  - 佛羅倫斯烏菲茲周邊
  - 威尼斯聖馬可廣場、水上巴士
  - 米蘭大教堂廣場
- 建議：包包背前面、手機不放後口袋、重要證件放飯店保險箱
